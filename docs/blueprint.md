# **App Name**: VozGenius

## Core Features:

- Painel Principal: Exibir um painel com informações gerais e acesso rápido às funcionalidades.
- Gravação de Voz: Permitir aos usuários gravar suas próprias vozes para uso posterior.
- Geração de Vozes: Integrar um gerador de vozes de texto para fala (TTS) alimentado por IA.
- Geração de Texto: Possibilitar a geração de textos por meio de inteligência artificial como uma ferramenta auxiliar.
- Lista de Vozes: Apresentar uma lista de vozes disponíveis para uso no gerador de texto para fala.
- Meus Pedidos: Listar os pedidos efetuados com status e informações de cada pedido.
- Créditos e Pacotes: Sistema de gerenciamento de créditos com opções de compra de pacotes.

## Style Guidelines:

- Primary color: Dark blue (#1E3A8A) to evoke trust and professionalism.
- Background color: Light gray (#F5F5F5) for a clean, modern look.
- Accent color: Orange (#EA580C) for calls to action and important highlights.
- Headline font: 'Poppins', a geometric sans-serif font, offering a precise, contemporary, and fashionable aesthetic that aligns well with the technological aspects of voice generation.
- Body font: 'PT Sans', a humanist sans-serif which balances a modern style with warmth, ensuring readability and approachability of content for users.
- Clean and modern icons to represent various actions and features.
- Use a grid-based layout to ensure consistency and readability across all pages.
- Subtle transitions and animations to improve user experience and provide feedback.
